# Authentication Policies

Authentication policies define the security rules that determine how users must authenticate.

Policies allow organizations to enforce security requirements depending on context.

## Policy Factors

Authentication policies may depend on:

- user location
- device type
- login time
- network environment
- security risk level

For example:

A system may require **multifactor authentication** if a login attempt originates from an unfamiliar location.
