# Authentication Overview

Authentication is the process of verifying a user's identity before granting access to a system.

Traditionally, authentication relied on usernames and passwords. Modern systems now incorporate stronger identity verification methods such as biometrics, passkeys, and multifactor authentication.

Authentication ensures that only verified users can access protected systems.

## Key Objectives

Authentication systems aim to:

- verify user identity
- prevent unauthorized access
- protect sensitive data
- maintain system integrity

After authentication is successfully completed, the system proceeds to authorization to determine the user's permissions.
